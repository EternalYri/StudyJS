Переменные и типы данных в JS

Типы данных

В JavaScript имеется 8 типов данных 6 первых из списка являются примитивными

  • Number - для любых чисел: целочисленных или чисел с плавающей точкой; целочисленные значения
  • String - для строк. Строка может содержать ноль или больше символов, нет отдельного символьного типа.
  • Boolean - для true/false
  • Null - для неизвестных значений – отдельный тип, имеющий одно значение null
  • Bigint для целых чисел произвольной длины.
  • Undefined - для неприсвоенных значений – отдельный тип, имеющий одно значение undefined.
  • Symbol - для уникальных идентификаторов.
  • Object для более сложных структур данных.

Для проверки типов днных используется опретор typeoff(х). Типы Null и Undefined являются пустыми значениями. Null - явно присвоенное пустое значение, а Undefined не присвоенное значение по умочанию.

Переменные в JS

В JS существует три способа объявить переменную:

  • var - не имеет блочной видимости, иными словами переменная объявленная таким способом будет видна через любые конструкции кроме функций, использовать такой способ в функции то она будет локальной перемернной внутри этой функции
  • let - более современный способ объявление перемернной для него существует блочная область видимости, иными словамми используя такой способ объявления переменная будет видна и использоваться только внутри блока например конструкции if, switch, циклах, функциях
  • const - похож на let за одним исключением, ее значение нельзя перезаписать и зачастую используется для объявления глобальных переменных
